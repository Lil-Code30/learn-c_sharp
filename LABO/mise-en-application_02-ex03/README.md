# Exercice 03 : La surcharges des opérateurs
En mettant en applications les concepts de surcharge des opérateurs, il est demandé de :
1. Définir la classe Durée avec les attributs suivants :
    - Heures : entier,
    - Minutes : entier,
    - Secondes : entier.
2. Mettre en place à l'aide de la surcharge d'opérateur les opérations suivantes :
    - Arithmétiques : addition et la soustraction entre deux durées,
    - Comparaison : égalité, inégalité, supériorité, infériorité, supériorité ou égalité, infériorité ou égalité.
