# Exercice 01 : Gestion des fichiers CSV (sans utiliser de bibliothèque)

À partir des notions apprises en classe, il est demandé de :
	1. Écrire deux fonctions permettant de :
		a	. Ecrire des données dans un fichier .csv
		b	. Lire les données à partir d’un fichier .csv
	2. Utiliser des tableaux associatifs ainsi que les librairies et méthodes suivantes :
		a	. Librairie : using System.IO;
		b	. Méthode : .Split(‘,’); (dans le cas où ; est utilisé comme délimiteur, adaptez le séparateur).
		c	. Méthode : .ToList();
	3. Veiller à produire un affichage clair, bien organisé et esthétiquement agréable.
