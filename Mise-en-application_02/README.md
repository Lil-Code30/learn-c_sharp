# Exercice 02 : « Les permissions & les constructeurs »

## Première partie :

Définir une nouvelle classe Étudiant avec les attributs : notes & moyenne.

Ajouter les méthodes suivantes :

1. Entrer les notes,
2. Calculer la moyenne,
3. Afficher les notes.

Appliquer sur chacun des méthodes et attributs les restrictions adéquates.

## Deuxième partie :

1. Rendre la classe Étudiant statique,
2. Définir les attributs & les méthodes adéquates en statique,
3. Utiliser cette classe afin de calculer la moyenne.


## Troisième partie :

Ajouter un constructeur à la classe Étudiant, Le constructeur ajouté servira à initialiser deux nouveaux attributs :

1. Le nom de l’étudiant,
2. Le prénom de l’étudiant.

Après avoir instancié un objet d’Étudiant, vérifier les valeurs des deux attributs.

## Quatrième partie :

Définir une nouvelle classe Date avec les attributs : jour, mois & année.

1. Faire en sorte que jour soit en lecture seule,
2. Faire en sorte que mois soit en écriture seule,
3. Faire en sorte qu’année soit en lecture & écriture.

Après avoir instancié un objet de Date, tester l’accès aux différents attributs.
