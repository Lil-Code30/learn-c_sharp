# Exercice 01 : « Introduction au paradigme orienté objet »

En se basant sur les principes de l’orientée objet, il est demandé de :

1. Créer une classe Facture contenant :
	a. Attributs : Client, Vendeur & Articles
	b. Méthodes : Calcul prix total, ajouter un Article & Afficher

2. Créer une classe Article contenant :
	a. Attributs : titre, quantité & prix unitaire
	b. Méthodes : Afficher

3. Ecrire un programme permettant de tester les classes ci-dessus
