# Exercice 02 : Les classes abstraites et interfaces

En mettant en application les concepts des classes abstraites et des interfaces, il est demandé de réaliser un programme pour modéliser le transport de marchandise avec les spécifications suivantes :
- Une marchandise doit comporter un poids et un volume,
- Le tarif d'un transport est le produit d'un prix unitaire par l'encombrement de la marchandise,
- Pour le transport aérien, l'encombrement est le poids de la marchandise. Le prix unitaire sera de 10,
- Si on veut un transport aérien urgent, il faut payer deux fois plus cher que le tarif normal, 
- Pour le transport routier, l'encombrement est le volume de la marchandise. Le prix unitaire sera de 2.
