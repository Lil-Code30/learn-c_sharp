# Exercice 03 : Gestion des fichiers JSON (en utilisant une bibliothèque)

Sur la base de ce qui a été fait en cours en lien avec la gestion des fichiers JSON avec la bibliothèque Json.NET, il vous est demandé de :

- Installer la bibliothèque [Json.NET](https://www.newtonsoft.com/json)  via le gestionnaire de paquets NuGet.
- Définir les entités avec les bonnes propriétés et permissions d’accès.
- Se référer à la documentation officielle pour respecter les standards de gestion des fichiers JSON.
- Implémenter un menu permettant
  - D’importer des données depuis un fichier JSON.
  - D’exporter une liste d’objets en fichier JSON.
  - De modifier et mettre à jour un fichier JSON existant.
