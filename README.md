## ğŸ¯ Topics Covered

### Core C# Concepts
- Basic arithmetic operations and data types
- Control flow and loops
- User input and output
- Type conversion

### Object-Oriented Programming (OOP)
- **Encapsulation**: Data hiding and access control
- **Abstraction**: Simplifying complexity through interfaces and abstract classes
- **Inheritance**: Code reuse and class hierarchies
- **Polymorphism**: Method overloading and overriding

### C# Specific Topics
- Access modifiers (`public`, `private`, `protected`, `internal`, etc.)
- Properties and accessors (getters/setters)
- `const` vs `readonly`
- Class vs instance methods and attributes
- Primary constructors and records
- Visual diagrams for access modifiers

## ğŸš€ Getting Started

### Prerequisites

- [.NET SDK](https://dotnet.microsoft.com/download) (version 6.0 or later recommended)
- An IDE or text editor:
  - [Visual Studio](https://visualstudio.microsoft.com/)
  - [Visual Studio Code](https://code.visualstudio.com/) with C# extension
  - [JetBrains Rider](https://www.jetbrains.com/rider/)

### Running the Projects

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd learn-c_sharp
   ```

2. **Run the basic exercises**
   ```bash
   cd allExercises
   dotnet run
   ```

3. **Run the Bank System tutorial**
   ```bash
   cd tutos/BankSystem
   dotnet run
   ```

4. **Build a solution**
   ```bash
   dotnet build allExercises.sln
   # or
   dotnet build BankSystem.sln
   ```
