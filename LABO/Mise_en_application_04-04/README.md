# Exercice 04 : Conversion entre JSON et CSV

En s'appuyant sur les exercices précédents, développez un programme permettant la conversion de fichiers CSV en JSON, et inversement.

- Installer les bibliothèques CsvHelper et Json.NET via NuGet.
- Implémenter un menu permettant de :
    - Importer et exporter des fichiers CSV et JSON.
    - Convertir un fichier CSV en JSON. 
    - Convertir un fichier JSON en CSV.
- Tester la conversion avec plusieurs jeux de données et observer d’éventuelles pertes d’informations lors du passage d’un format à l’autre.
